INTRODUCCION A LA PLATAFORMA DE PRESUPUESTOS



Estructura básica de nuestro sistema de gestión de presupuestos:

Gestión de cuentas de usuario

Hay tres tipos de cuentas de usuario: administrador, gerente y empleado. Cada uno de estos roles tiene acceso a diferentes funcionalidades.


Gestión de productos

Los productos se pueden agregar de dos maneras: automáticamente, con una plantilla en formato excel, o manualmente.




Gestión de Clientes y Contactos.

Los Clientes y Contactos se pueden agregar de dos formas: automáticamente, con una plantilla en formato excel, o manualmente.





Gestion de presupuestos

Un asistente guía al usuario a través de los elementos necesarios para crear una cuota.:
  • 1. Cliente y contacto + anotaciones.
  • 2. Productos + asignación de usuarios + anotaciones.
  • 3. Revisión de la cuota. Disponible para el cliente en formato PDF.

La lista de presupuestos se puede filtrar, reutilizar, editar, etc.





Historico de presupuestos

Todos los eventos realizados en un mismo presupuesto quedan registrados para luego ser consultados en el historial; aquí puedes ver cualquier cambio en el estado de la cuota.

Gráficos

Las cuentas de usuario administrador y manager tienen acceso a una herramienta especial para ver la evolución de los presupuestos, descargarlos en formato excel y filtrar por varios criterios: fecha, usuario, cliente, estado, etc.
